# InvoiceFlow PWA

Une application moderne et intuitive pour générer des factures professionnelles en quelques secondes.

## Fonctionnalités
- **Mode PWA :** Installable sur votre appareil pour un accès rapide.
- **Conformité :** Gestion automatique des mentions légales (TVA 0%).
- **Flexibilité :** Saisie libre de la TVA, des dates et du logo.
- **Sécurité :** Pas de stockage de données, tout reste sur votre appareil.
- **Export :** Génération instantanée de PDF haute qualité.

## Installation Locale
1. Clonez le dépôt.
2. `npm install`
3. `npm run dev`

## Utilisation
1. Remplissez les informations de votre entreprise.
2. Ajoutez votre logo.
3. Saisissez les coordonnées du client et les lignes de services.
4. Cliquez sur "Télécharger PDF".
